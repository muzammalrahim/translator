## Project Setup

Run composer install
<br/>
On cmd run the command: php artisan key:generate
<br/>
Rename .env.example to .env
<br/>
Add GOOGLE_API_KEY in .env
<br />
Run the command: php artisan serve OR use localhost to run the project
